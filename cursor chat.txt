AH01909 SSL warning is unrelated to /api 404.

Include conf/extra/httpd-vhosts.conf
If it is disabled, Apache ignores httpd-vhosts.conf.

Put the alias in the file that is definitely loaded

Alias /api "D:/Workability/AISCO/AISCO/api/public"

LoadModule rewrite_module modules/mod_rewrite.so
